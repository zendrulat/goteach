Here are golang commands

To run a go program there a few different ways

1. Go Run 


go run main.go
simply runs the program

2. Go build 


go build main.go
Builds out the folders

3. Go install 


go install main.go
makes an excutable binary

4. Go fmt 


gofmt main.go
formats your code

For Go debugging

4. debugging 


//get package
go get github.com/mailgun/godebug

//insert a breakpoint
_ = "breakpoint"

//then
godebug run main.go
